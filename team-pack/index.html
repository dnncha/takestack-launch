A repeatable assignment, not a video-editing class.

The team pack is for groups that need clear capture instructions and a fast local assembly workflow before a rehearsal clip, concert promo, class project, or worship announcement goes out.

  1. Send one guide

    Share the same guide track, count-in, or rehearsal recording so every take follows one timing reference.

  2. Ask for one clear cue

    Have each singer or player include a clap, lyric, count, or first note that can be lined up quickly.

  3. Collect vertical phone videos

    Keep source files local and portrait-oriented when the finished video is for Reels, TikTok, or Shorts.

  4. Stack and export on Mac

    Import the takes, choose the guide, line up the cue, use a music-friendly layout, balance sound, and export MP4.
