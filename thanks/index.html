What happens next.

Use this page as the Stripe confirmation handoff. It tells buyers what to expect before the private delivery email arrives, points to the in-editor First export panel, and gives group buyers the setup details to send back. If the buyer needs instant access before the delivery reply arrives, the unlisted direct-beta download handoff is available at /founder-download.

  1. Confirm receipt

    Look for your Stripe receipt email. The same address is used for the app delivery and founder support.

  2. Watch for delivery

    The founder license or team pack is delivered manually with the Mac app zip, checksum, and install guide. The unlisted buyer download page can be used for instant direct-beta access when support points you there.

  3. Open carefully

    The direct beta may show a macOS developer warning. Use the included install guide and checksum before opening.

  4. Use the First export panel

    Choose Project > Open Demo Project, then use the visible First export panel above the preview for Preview, Line up, Export MP4, Copy Checklist, or Send Export Report...

  5. Report or reply if blocked

    After the demo export, use Support > Send Export Report... if you are willing to share source and export outcome. First 10 paid buyers can reply before 6pm Ireland time for a same-day founder export check by email. Reply to the delivery email for setup help or a 14-day founder refund if the beta cannot get you to an export.

Founder license, team pack, or concierge sprint?

Solo buyers should start with the demo project and one real music-video export. Team-pack buyers should reply to the receipt email with the group details. Deadline buyers should use the concierge sprint path so the first-video support scope is clear before kickoff.

  • Founder license: send the app to the Stripe receipt email and start with Project > Open Demo Project plus the visible First export panel.
  • First-buyer export check: for the first 10 paid buyers, reply to the Stripe receipt with the source setup, demo-export blocker, or next-step question. This is email help, not done-for-you editing, live setup, or automatic sync.
  • Team pack: reply with group size, Macs available, first-video deadline, and who will edit the final stack.
  • Need more seats? Read the $149 team-pack details or use Buy the 5-seat team pack.
  • Need deadline help? Read the $499 concierge sprint scope or use Buy the concierge sprint.
  • Need instant access? Use the unlisted direct-beta download handoff only if you bought TakeStack or received a seed-license reply.
  • Refund path: 14-day founder refund if the beta cannot get the buyer to a real music-video export.
  • Privacy: TakeStack works with local files and does not upload buyer videos in the MVP.
